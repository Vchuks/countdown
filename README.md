# CountDown Application

An application where you can add events and you can see the day, time, hour or seconds left before the events.


## Tech stack

- **React** (Vite)
- **Tailwind Css** 


## Getting started

```bash
# 1. Clone the repo
git clone https://github.com/Vchuks/countdown.git
cd countdown

# 2. Install dependencies
npm install

# 3. Run locally
npm run dev
```
